# tn-core

Python bindings for the `tn-core` Rust runtime — the bedrock layer
that handles emit, read, signing, and chain verification. Built with
PyO3 + maturin.

Imported by the `tn-protocol` SDK; you usually don't install this
package directly.

```bash
pip install tn-core   # pulled in automatically by `pip install tn-protocol`
```

Source: https://github.com/cyaxios/tn-proto
