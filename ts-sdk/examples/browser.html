Every action, a transaction. Run these examples live in the browser.

Init

Create a ceremony. Your device gets a unique attested identity.

import * as tn from "@tnproto/sdk/browser";

await tn.init();
console.log(tn.did()); // did:key:z...
Click Run to initialize a TN session.

Emit

Write attested log entries. Every call is signed and chained.

tn.info("payment.processed", { amount: 99.00, currency: "USD" });
tn.warning("rate.limit.approaching", { threshold: 0.92 });
tn.error("auth.failed", { ip: "10.0.0.1", attempts: 3 });
tn.debug("cache.miss", { key: "user:42:profile" });
Init first, then run.

Read

Read back typed Entry objects from the session log.

for (const entry of tn.read()) {
  console.log(
    entry.level,
    entry.event_type,
    entry.sequence,
    entry.fields
  );
}
Emit some entries first.

Context

Scope fields across a group of events without repeating them.

tn.setContext({ session_id: "sess_abc", user_id: "u_123" });

tn.info("page.view", { path: "/dashboard" });
tn.info("action.click", { element: "export_button" });

tn.clearContext(); // stop attaching session fields
Init first, then run.

Decode

Any source

Turn any TN-formatted JSON object into a typed Entry — fetch, SSE, wherever.

import { decodeEntry } from "@tnproto/sdk/browser";

// works on any JS object — not just from tn.read()
const entry = decodeEntry(envelope);
console.log(entry.event_type, entry.fields);
Paste or edit a TN envelope above, then click Decode.

Chain

Every entry carries a hash chain and Ed25519 signature. Nothing is hand-wavy.

const entries = tn.read();
const e = entries[0];

console.log("seq      ", e.sequence);
console.log("row_hash ", e.row_hash);
console.log("prev_hash", e.prev_hash);
console.log("signature", e.signature);
Emit some entries first.
