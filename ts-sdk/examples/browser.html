Init

Create a ceremony. Your device gets a unique attested identity.

import * as tn from "@tnproto/sdk/browser";

await tn.init();
console.log(tn.did()); // did:key:z...

Emit

Write attested log entries at any severity. Every call is signed and chained.

tn.info("payment.processed", { amount: 99.00, currency: "USD" });
tn.warning("rate.limit.approaching", { threshold: 0.92 });
tn.error("auth.failed", { ip: "10.0.0.1", attempts: 3 });
tn.debug("cache.miss", { key: "user:42:profile" });
Init first, then run.

Read

Read back typed Entry objects from the session log.

for (const entry of tn.read()) {
  console.log(entry.level, entry.event_type, entry.sequence, entry.fields);
}
Emit some entries first.

Context

Scope fields across a group of events without repeating them.

tn.setContext({ session_id: "sess_abc", user_id: "u_123" });

tn.info("page.view", { path: "/dashboard" });
tn.info("action.click", { element: "export_button" });

tn.clearContext(); // stop attaching session fields
Init first, then run.

Decode

Any source

Turn any TN-formatted JSON into a typed Entry — fetch, SSE, vault API, anywhere.

import { decodeEntry } from "@tnproto/sdk/browser";

const entry = decodeEntry(envelope); // any JS object
console.log(entry.event_type, entry.fields);
Edit the envelope above, then click Decode.

Chain

Every entry carries a hash chain and Ed25519 signature.

const [e] = tn.read();

console.log("seq      ", e.sequence);
console.log("row_hash ", e.row_hash);
console.log("prev_hash", e.prev_hash);
console.log("signature", e.signature);

Receive

Database auth

A stream of entries sealed for your key. The encryption is the authorization.

// fetch any endpoint that returns NDJSON TN entries
const res = await fetch(url);
const text = await res.text();

for (const line of text.split("\n").filter(Boolean)) {
  const entry = decodeEntry(JSON.parse(line), keystore);
  // entry.hidden_groups.length > 0  →  sealed, no kit
  // entry.fields.account_balance    →  decrypted with your kit
}

Keys

Multi-recipient

Each group is sealed under a different publisher. Each kit only opens its own group.

// Publisher side — two groups, two recipients
const clientPub     = new BtnPublisher(null);
const compliancePub = new BtnPublisher(null);

const clientKit     = clientPub.mint();     // issued to the client
const complianceKit = compliancePub.mint(); // issued to compliance

// Each entry carries two sealed groups
entry["client"]     = { ciphertext: seal(clientPub,     { balance, action }) };
entry["compliance"] = { ciphertext: seal(compliancePub, { risk_score, flag }) };
